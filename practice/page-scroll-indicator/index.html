Scroll Indicator

Notes

This page shows a green indicator that signifies how far the user has scrolled. I was inspired to do this page because I wanted to do a page indicator inspired by this other design I was practicing. This looks to me like a very simple version of that thing. But I might be totally wrong. Nevertheless I will learn someting from this.

The top of this page contains a div called progress-container that contains an empty div called progress-bar

The progress-container is the dark gray thing and the progress-bar is the green thing. Initially its width is set to 0.

The main logic of this code is based on utilizing window.body.scroll

So getting a fish would be learning this trick. But learning to fish would be taking a look at Window.scroll() documentation on MDN

Algorithm

1. Scrolled = Get the user's scroll position in pixels
2. Height = Get the height of the content in pixels
3. ProgressInPercentage = Scrolled / Height * 100;
4. Set width of progress-bar = ProgressInPercentage

I just used pre tag to write the above algorithm and it really messed up the mobile view. I've changed it to use code tag
